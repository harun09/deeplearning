Kernel Density Estimates

Kernel Density Estimation (KDE) merupakan suatu cara untuk memperkirakan fungsi probabilitas kepadatan dari sebuah variabel acak kontinu. Hal ini digunakan untuk analisis non-parametric. Pengaturan hist flag dengan nilai false dalam distplot akan menghasilkan plot estimasi kepadatan kernel. Di seaborn, kita dapat mem-plot Kernel Density plot dengan menggunakan jointplot().

In [20]:

sb.distplot(df['petal_length'],hist=False)
plt.show()

Plotting Bivariate Distribution

Distribusi bivariate digunakan untuk menentukan hubungan antara dua variabel. Ini terutama mengatasi hubungan antara dua variabel dan bagaimana satu variabel berperilaku dengan yang lain. Cara terbaik untuk menganalisis distribusi bivariate di seaborn adalah dengan menggunakan fungsi jointplot ( ). Jointplot menciptakan sebuah figur multi-panel yang proyeksikan hubungan bivariate antara dua variabel dan juga distribusi univariate untuk setiap variabel pada poros terpisah.

Scatter Plot

Scatter plot adalah cara yang paling mudah untuk memvisualisasikan distribusi di mana setiap pengamatan diwakili dalam plot dua dimensi melalui sumbu x dan y.

In [28]:
import pandas as pd
import seaborn as sb
from matplotlib import pyplot as plt
df = sb.load_dataset('iris')
sb.jointplot(x = 'petal_length',y = 'petal_width',data = df,kind = 'scatter')
plt.show()

Gambar di atas menunjukkan hubungan antara petal_length dan petal_width dalam data iris.Tren pada plot mengatakan bahwa korelasi positif antara variabel.

Visualizing Pairwise Relationship

Datasets dalam penelitian yang bersifat real-time memiliki banyak variable. Dalam kasus ini, hubungan ntara masing-masing dan setiap variabel harus dianalisis. Distribusi bivariate untuk kombinasi (n, 2) ini merupakan salah satu kombinasi yang sangat kompleks dan menghabiskan waktu. Untuk mem-plot beberapa penyebaran dalam sebuah kumpulan data pairwise bivariate, anda dapat menggunakan fungsi pairplot ( ). Hal ini menunjukkan hubungan untuk kombinasi yang berdampingan dari variabel (n, 2) dalam sebuah dataframe sebagai sebuah matrix plot dan plot diagonal adalah plot univariate.

In [29]:
import pandas as pd
import seaborn as sb
from matplotlib import pyplot as plt
df = sb.load_dataset('iris')
sb.set_style("ticks")
sb.pairplot(df,hue = 'species',diag_kind = "kde",kind = "scatter",palette = "husl")
plt.show()
